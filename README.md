# SEO Rank Tracker

An AI-powered SEO analytics platform that helps businesses, developers, and marketers audit websites, monitor SEO health, and improve search visibility through actionable, data-driven insights.

## ✨ Features

- Instant SEO audits with a comprehensive health score
- AI-generated insights and optimization recommendations
- Keyword and on-page SEO analysis
- Site-wide website crawling and issue detection
- Competitor comparison dashboard
- Core Web Vitals and performance monitoring
- Audit history with trend visualization
- Professional PDF report generation
- Secure authentication with saved reports
- Fully responsive interface with light and dark themes

## 🛠️ Tech Stack

- **Frontend:** React 19, Vite, TypeScript, Tailwind CSS, React Router, Recharts
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Authentication:** JWT, bcrypt
- **AI & APIs:** Google Gemini API, Google PageSpeed Insights API

## 🚀 Getting Started

```bash
git clone https://github.com/<your-username>/RankIntel.git
cd RankIntel
npm install
npm run dev
```

## 🎯 Vision

SEO Rank Tracker is designed to provide enterprise-grade SEO analysis in a modern, intuitive interface, empowering users to make informed decisions and continuously improve their online presence.
